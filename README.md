# Customer Segmentation using Machine Learning

## 📖 Project Overview
This project performs Customer Segmentation using the Mall Customers dataset.  
The goal is to group customers based on their purchasing behavior using clustering techniques.

## 📂 Dataset
- Mall_Customers.csv
- Dataset contains customer information such as:
  - Customer ID
  - Gender
  - Age
  - Annual Income
  - Spending Score

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🤖 Machine Learning Algorithm
- K-Means Clustering

## 📊 Project Steps
1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Applying K-Means Clustering
5. Visualizing Customer Segments

## 📈 Output
Customers are divided into different clusters based on:
- Annual Income
- Spending Score

These clusters help businesses:
- Identify high-value customers
- Create targeted marketing strategies
- Improve business decision-making

## ▶️ How to Run
1. Open `Customer_Segmentation.ipynb`
2. Run all cells in Jupyter Notebook or Google Colab
3. View clustering results and visualizations
